# hubot-kvv

A hubot script that access http://live.kvv.de/

See [`src/kvv.coffee`](src/kvv.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-kvv --save`

Then add **hubot-kvv** to your `external-scripts.json`:

```json
[
  "hubot-kvv"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
